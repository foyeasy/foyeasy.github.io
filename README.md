# Mohammad Ramjan Hossain — Academic Portfolio

[![Live Site](https://img.shields.io/badge/Live-foyeasy.github.io-2e7d52?style=flat-square&logo=github)](https://foyeasy.github.io)
[![GitHub Pages](https://img.shields.io/badge/Hosted-GitHub_Pages-181717?style=flat-square)](https://pages.github.com)

A clean, responsive academic portfolio showcasing research, publications, and engineering projects in mechanical engineering, additive manufacturing, and computational modeling.

## 🌐 Live Demo
🔗 **[foyeasy.github.io](https://foyeasy.github.io)**

## ✨ Features
- ⚡ **Zero Dependencies**: Built with pure HTML5, CSS3, and Vanilla JavaScript — no frameworks or build steps
- 🔄 **Smooth Interactions**: Scroll-triggered animations, interactive carousels, and hover effects
- 🧩 **Modular Architecture**: Reusable component classes (`.r-card`, `.pub-card`, `.tl-item`) for easy content updates
- 🔍 **SEO & Accessibility**: Semantic HTML structure, ARIA labels, and optimized meta tags

## 🛠 Tech Stack
| Category | Tools & Technologies |
|----------|----------------------|
| **Frontend** | HTML5, CSS3, Vanilla JavaScript |
| **Typography** | Google Fonts: Playfair Display, Nunito Sans, DM Mono |
| **Animations** | CSS Transitions, Intersection Observer API |
| **Icons** | Inline SVGs + PNG fallbacks |
| **Hosting** | GitHub Pages |
